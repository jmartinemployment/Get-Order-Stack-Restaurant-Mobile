# POS App Assets

This folder should contain:

1. `icon.png` - App icon (1024x1024)
2. `splash.png` - Splash screen image (1284x2778)
3. `adaptive-icon.png` - Android adaptive icon foreground (1024x1024)

For now, Expo will use default placeholders.

To generate proper icons:
1. Create a 1024x1024 PNG with your logo
2. Use https://easyappicon.com/ to generate all sizes
